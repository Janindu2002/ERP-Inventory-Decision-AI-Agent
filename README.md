# ERP Inventory Decision AI Agent

## Overview
The **ERP Inventory Decision AI Agent** is an AI-powered assistant that helps managers analyze inventory data and make informed business decisions. The system reads inventory reports (CSV or PDF), evaluates stock levels, sales trends, and supplier information, and provides actionable insights.

It can also **automatically generate reorder invoices** when products fall below the required stock level.

This project demonstrates the combination of **Business Analysis, ERP workflow understanding, data analysis, and AI-assisted decision support**.

---

## Key Capabilities

- Identify products with **low stock levels**
- Recommend **items that need reordering**
- Analyze **product demand based on sales data**
- Compare **suppliers by cost and delivery speed**
- Generate **automatic reorder invoices**
- Provide **AI-driven insights from inventory reports**


---

## Technologies Used

- **Python** – Core application development  
- **Pandas** – Inventory data analysis  
- **LangChain** – AI agent framework  
- **LM Studio** – Local LLM for AI reasoning  
- **ReportLab** – Reorder invoice generation  
- **Jupyter Notebook** – Data exploration and analysis  

---

## Project Purpose

This project demonstrates how AI can support **ERP inventory decision-making**, reduce manual analysis, and help businesses prevent stock shortages through **data-driven insights and automated workflows**.
